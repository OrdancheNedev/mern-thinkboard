# MERN Thinkboard (Note App)

A full-stack Note application built using the MERN (MongoDB, Express.js, React, Node.js) stack. This project allows users to create, read, update, and delete notes in a responsive dashboard interface.

## 🚀 Live Demo

You can view the deployed application here: [MERN Thinkboard](https://mern-thinkboard-ck7r.onrender.com/)

## ✨ Features

- Full CRUD functionality for managing notes.
- Responsive UI designed for both desktop and mobile devices.
- Production deployment on Render.

## 🛠️ Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Atlas)
- **Hosting/Deployment:** Render
